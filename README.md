## 🔸 내용
- Cola Cola 로고

- 음료수 종류 6캔
  - Original Cola
  - Violet Cola
  - Yellow Cola
  - Cool Cola
  - Green Cola
  - Orange Cola

- 각 음료 가격 1000원 동일 

- 잔액
- 거스름돈 반환

- 입금
- 입금액 입력

- 선택한 음료 현황
- 획득한 음료 현황

- 획득
- 소지금
- 총금액
